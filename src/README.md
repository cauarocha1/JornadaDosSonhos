# Execucao rapida

## Dependencias

```bash
pip install streamlit requests
```

## Ollama (opcional)

```bash
ollama pull gpt-oss
ollama serve
```

## Rodar aplicacao

```bash
streamlit run src/app.py
```

## Endereco

`http://localhost:8501`

## Dica

Se o modelo demorar para responder, aumente `Timeout de leitura (s)` na barra lateral.
